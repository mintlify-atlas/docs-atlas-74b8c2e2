## Quick Example

Send a notification when your long-running job completes:

```bash
# Run your job and notify when done
./train_model.sh && whecho "Model training complete! 🎉"
```

Or use it from Python:

```python
from whecho.whecho import whecho_simple

# Send notification from your script
whecho_simple("Data processing finished!")
```

## Get Started in Minutes

<Steps>
  <Step title="Install whecho">
    Install via pip in seconds
    
    ```bash
    pip install whecho
    ```
  </Step>
  
  <Step title="Configure your webhook">
    Set up your webhook URL once
    
    ```bash
    whecho --init
    ```
  </Step>
  
  <Step title="Send your first message">
    Start receiving notifications
    
    ```bash
    whecho "Hello from the command line!"
    ```
  </Step>
</Steps>
